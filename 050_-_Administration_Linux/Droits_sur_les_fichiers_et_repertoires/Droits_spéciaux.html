Droits spéciaux

Créée le jeudi 15 mai 2025

en troisième position de chaque

rwSrwxrwT

Users : s S x : setuid
Groups : s S x : setgid
Others : t T x : stickybit

Le droit S et T majuscule désactive le x, le s et t conserve le x.

setuid :

fichiers ordinaires : fichiers exécutables, l'exécution se fera avec les privilèges de l'utilisateur propriétaire du fichier
fichiers de type dossier : ne veut rien dire

setgid :

fichiers ordinaires : fichiers exécutables, groupe propriétaire
fichiers de type dossier : Héritage du groupe propriétaire (du dossier) sur tous les fichiers qui y seront créés

stickybit :

fichiers ordinaires : fichiers conservé en mémoire
fichiers de type dossier : seul le propriétaire d'un fichier peut supprimer celui-ci

Activation :

chmod u+s
chmod g+s
chmod +t
ou
chmod u-s
etc

En octale :

On ajoute un chiffre avant le trio de chiffres : x755

s setuid = 4
s setgid = 2
stickybit = 1

Donc si je veux placer t : 1755
Si je veux placer le setgid et le stickybit : 3755
Si je veux retirer les droits spéciaux : 0755
Etc..
