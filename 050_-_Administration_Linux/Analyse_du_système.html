Analyse du système

outil neofetch :
information sur le système

Distribution :

lsb_release -a (Debian)

Sur une autre distribution que Debian :

ls /etc/*version
ls /etc/*release
hostnamectl

Noyau Linux :

hostnamectl
uname -a

Processeur :

lscpu

Lister les périphériques matériels :

lspci

Usage de la mémoire vive / swap :

free -h

Afficher les périphériques de stockage :

lsblk
fdisk -l

Informations sur les volumes LVM :

vgs
lvs

Afficher le système de fichiers de partitions ou lv :

blkid
lsblk -f

Afficher l'espace occupé/libre des systèmes de fichiers montés :

df -h

Espace occupé par les fichiers situés dans un dossier :

du -sh

Afficher les processus :

ps -ef
ps aux
top
pgrep -l <nom> (recherche de processus par le nom)

Envoi de signaux à un processus :

kill <signal> PID
