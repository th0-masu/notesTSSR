Commandes

Créée le lundi 28 avril 2025

Commande Paramètres Rôle Notes
tty   Selectionne un terminal ou voit sur lequel on est connecté tty : terminal / pts : terminal virtuel (session SSH par exemple)
time <cmd> Affichage du temps d'execution d'un processus time ./trait_ventes.sh
passwd [OPTIONS] [LOGIN] Changement de mot de passe Sans option, la commande passwd permet de modifier le mot de passe, si le login n'est pas fourni la commande se base sur l'utilisateur courant.
!? fich Rappel de la dernière commande content une chaine Relance une commande contenant fich
rm -r -i Supprime un fichier ou un dossier -r : Résursif , -i : confirmation de suppression
cp <SOURCE> <DESTINATION> Copie de fichier -r : récursif (copie le dossier et tout son contenu)
ls -l , -R, -a Lister les fichiers et dossiers du répertoire -R - Récursif
pwd   Affiche le répertoire courant print working directory
ln -s , <source> <lien> Créer des liens Lien physique ou lien symbolique
id   Affiche l'user ID (uid), le group ID (gid) Les utilisateurs et groupes sont gérés avec des numéros. On peut demander un autre utilisateur (id user02)
tail -f Affiche les dernières entrées d'un fichier -f affiche en continu, utile pour actualiser les dernières entrées d'un log par exemple
date   Affiche la date du jour En contexte admin #, peut modifier la date
wall   Affiche un message sur tous les shells utilisateurs actifs echo "message" | wall
cd   Se déplacer dans la hiérarchie de fichiers du système change directory
kill [-signal] <PID> Termine un processus PID : identification du processus à stopper 15 Terminaison logicielle - 2 Interruption (^C) - 1 Fin de session - 9 Arrêt inconditionnel
! 128 Relance une commande !com relance la précédente commande commençant par com, !27 relance la dernière commande numéro 27
man   Affiche le manuel d'une commande  
  -k "log" Rechercher une expression dans les titres des pages de man  
       
* --help Sur n'importe quel commande, affiche un résumé de l'aide  
       
       
set   Affiche toutes les variables  
env   Affiche uniquement les variables d'environnement  
history   Affiche l'historique des commandes tapées  
       
logname   Affiche le nom d'utilisateur  
who   Information sur les utilisateurs connectés  
  -H Heading, affichage de l’entête de colonnes.  
  -q Liste abrégée avec les noms et le nombre d'utilisateurs.  
  -T Indique par + ou – s'il est possible d'écrire sur le terminal.  
  -u Informations complémentaires sur les différentes connexions.  
who am i   N'affiche que les informations relatives à l'utilisateur  
  + format Format de visualisation de la date, composé de texte et de différents paramètres identifiés par le caractère %  
cal   Affiche le calendrier  
  05 2021 Affiche le calendrier d'une date spécifique, année etc  
       
       
       
mkdir   Créer un dossier  
       
touch   Création de fichier vide  
mv   Déplace ou renomme un fichier ou un dossier  
       
cat   Lire un fichier  
head   Afficher les premières entrées d'un fichier  
       
wc   Compte le nombre de caractères, de mots, de lignes... d'un fichier  
diff   Liste la différence entre 2 fichiers  
       
       
cat   Affichage de texte brut dans la console  
more   Affichage du texte par page  
less   Pareil que MORE mais avec quelques fonctionnalités en plus  
       
vi   Lance VI ou VIM  
  / En mode commande, lance la recherche  
  :wq Sauvegarder et quitter  
  :q Sortie sans sauvegarde  
  ! Force l'exit avec :q sans enregistrer  
  :nb :10 > se déplace à la ligne 10  
  [nb]← [nb]↑ etc se déplace de nb vers la direction  
  gg ou :0 Début du fichier  
  G ou :$ Fin du fichier  
  :set number Affiche les lignes :set nonumber pour désactiver  
  :set relativenumber Affiche les lignes relatif à la ligne en cours :set norelativenumber pour désactiver  
       
  dd Supression de ligne  
  x ou X supression de caractères, [nb]x supprime nb caractères  
  [nb]y(wW) Copie nombres de mots à partir de la position  
  [nb]yy    
       
ps   Affichage des processus de l'utilisateur courant  
  -fax Affiche les processus parents et enfants  
       
&   Utilisé pour lancer une commande en arrière-plan  
jobs -l Affiche les commandes en arrière plan. -l affiche les PID  
       
tar -c(reate) -f(ile)<destination.tar> <cible à archiver> -t affiche le contenu sans extraire  
  -x(extraction) -v(erbose) -f(ile) <fichier à extraire> -C(restauration) <Dossier cible restauration>    
  -c -z(active) la compression -v -f <destination.tar.gz> <cible à archiver> .tar.gz ou .tgz pour une archive compressée  
  -c -j(active compression bz2) -v -f <destination .tar.bz2> <cible à archiver> .bz2 un peu plus long mais meilleure compression  
       
ln -s /chemin/fichier/source.txt /chemin/fichier/destination.txt    
       
       
tee -a(ppend) <destination> Redirige sur le flux vers la sortie standard et vers un fichier
