M365 - Présentation

Microsoft 365 est une offre Saas – Software as a Service. Les logiciels sont installés sur un serveur distant et mis à disposition.

  • Gestion On-Premises = en interne

Gestion de tous les aspects de l'infrastructure; donc matériel et personnel

  • Gestion Infrastructure as a Service IaaS

On gère une partie (applications, les OS) et le service nous loue les serveurs, le stockage, le réseau

  • Gestion Platform as a Service PaaS

On gère les applications et la data et le service nous loue toute l'infrastructure (OS, Serveurs..)

  • Gestion Software as a Service SaaS

Tout est externalisé, même les applications. Le service nous loue une application.

  • Offre Cloud
  • Différentes applications (Exchange, Sharepoint, Teams..)
  • Multi-plateformes
  • Particuliers et entreprises
  • Moins de serveurs sur site
